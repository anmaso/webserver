WebServer Configurable Web Server

Welcome to the WebServer! This server is running and ready to handle requests.

Available Endpoints

GET /config - Get current configuration
PUT /config - Update configuration
GET /stats - Get server statistics
GET /ws - WebSocket endpoint for TUI

Testing Dynamic Endpoints

Try these default endpoints to test the dynamic behavior:

GET /api/error - Returns a 500 error
GET /api/delay - Returns a delayed response (2 seconds)
GET /api/flaky - Returns an error every 3rd request

Configuration

The server configuration is hot-reloadable. Modify the configuration file or use the /config endpoint to update settings.
